# Caso k = 2 — Problema Pesimista

Esta carpeta contiene el código asociado al análisis del Problema Pesimista
cuando se permiten hasta dos elecciones (k = 2).

Se incluyen:

- el programa dinámico que calcula la política óptima para parámetros
  de recompensa (q1, q2, q12),
- pruebas numéricas del programa dinámico,
- comparaciones entre el programa dinámico y el análisis continuo
  para distintos objetivos.
